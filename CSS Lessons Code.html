CSS Color Keywords:

Transparent Keyword:

CSS: div.ex1 { background-color: lightgreen; border: 2px solid black; padding: 5px; } div.ex2 { background-color: transparent; border: 2px solid black; margin-top: 10px; padding: 5px; } HTML: <div class="card2" style="background-color: rgb(145, 145, 240);"> <div class="ex1"><p>While the base background is light blue, this div's background color is light green.</p></div> <div class="ex2"><p>This div has the <i>transparent</i> value on its style value, thus having the same background as the previous div.</p></div> </div>

Currentcolor Keyword:

CSS: div.ex3 { color: blue; border: 5px solid currentcolor; padding: 5px; } HTML: <div class="card2"> <div class="ex3"><p>This div element has a blue text color. The border is also blue due to the currentcolor keyword.</p></div> </div>

Inherit Keyword:

CSS: div.ex4 { border: 2px solid red; } span { border: inherit; } HTML: <div class="card2"> <div class="ex4">In this, the <span>span element's</span> border will be inherited from the parent element</div> </div>
