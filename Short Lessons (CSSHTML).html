CSS Color Keywords:

This short lesson will teach you CSS Color Keywords such as tranparent, currentcolor, and inherent.

Transparent Keyword:

The transparent keyword is used to make a color transparent.

Currentcolor Keyword:

The currentcolor keyword is like a variable that holds the current value of the color property of an element.

This div element has a blue text color. The border is also blue due to the currentcolor keyword.

Inherit Keyword:

The inherent keyword, in the name itself, specifies that a property will inherit its value from its parent element.

In this, the span element's border will be inherited from the parent element

CSS Gradients:

This short lesson will teach you CSS Gradients. Gradients allow you to display smooth transitions between two or more colors.

There are three types of Gradients:

  • Linear Gradients - it goes up, down, left, right, or diagonal.
  • Radial Gradients - circular.
  • Conic Gradients - rotated around a point.

Linear Gradients:

Basic Syntax: background-image: linear-gradient(direction, color-stop1, color-stop2, ...);

CSS 2D Transform:

With CSS, you can move, rotate, scale, and skew elements. You can also create simple 2D animations with it.

Translate Method:

The translate(x-axis, y-axis) method moves an element from its current position.

This box is moved 100 pixels to the right and 25 pixels below.

Rotate Method:

The rotate(degree) method rotates an element clockwise or counter-clockwise according to a given degree.

This box rotated clockwise by 50 degrees.

Scale Method:

The scale(width, height) method increases or decreases the size of an element. If you only want to change only one size, use scaleX(width) for witdh or scaleY(height) for height.

This box is twice its original size.

Skew Method:

The skewX(degree) and skewY(degree) methods skews an element along the X-axis and Y-axis by the given angle. The shorthand version for this is skew(X-axis, Y-axis
